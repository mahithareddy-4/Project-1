# Multiple Disease Prediction System using Machine Learning

## Overview

This project is a healthcare-focused Machine Learning application that predicts the likelihood of multiple diseases using trained ML models.

The system provides prediction modules for:

* Diabetes Prediction
* Heart Disease Prediction
* Parkinson's Disease Prediction

An interactive Streamlit interface allows users to enter medical parameters and receive real-time prediction results.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* Streamlit

## Features

* Multiple disease prediction modules
* User-friendly Streamlit interface
* Real-time prediction results
* Machine Learning based healthcare analytics

## Project Structure

* app.py
* dataset/
* saved_models/
* requirements.txt

## How to Run

1. Install dependencies

pip install -r requirements.txt

2. Run the application

streamlit run app.py
